# RecyclerViewQuickAdapter

## 功能
- Header、Footer
- LoadMore
- EmptyView
- ItemAnimator

## 开始使用
```
allprojects {
    repositories {
        jcenter()
        maven{url 'https://dl.bintray.com/1109993488/maven'}
    }
}

dependencies {

    compile 'com.blingbling:RecyclerViewQuickAdapter:1.0.0'
}
```
